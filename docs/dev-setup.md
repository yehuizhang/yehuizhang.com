# Development Environment Setup

## Containers
```shell
# Start
docker-compose -p "zyh-gin-webapp" up --build --detach
# Stop
docker-compose -p "zyh-gin-webapp" down
```

- [Regis-Insight](https://redis.com/redis-enterprise/redis-insight/)
